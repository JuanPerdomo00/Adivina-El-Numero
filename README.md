# Adivina el numero
Esto es un programa de un reto que nos proporciono platzi. En le curso de python3


## run mac
```sh
# Ejecuta el script de python
zsh run.sh
```

## run Linux y windows
```
pip3 install -r requirements.txt
python3 script.py
```

## Usar el ejecutable (LINUX)
```
chmod 755 adivina_el_numero
./adivina_el_numero
```
